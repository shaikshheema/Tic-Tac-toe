# 🎮 Tic Tac Toe Game

A simple web-based Tic Tac Toe game built using **HTML**, **CSS**, and **JavaScript**.

You can play it live here:  
👉 [https://shaikshheema.github.io/tictactoe](https://shaikshheema.github.io/tictactoe)

- Stylish UI
- Fully responsive
